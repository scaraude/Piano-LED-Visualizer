If the automatic conversion does not suit your expectations, you can load your own musical notation.
The highlighted part of the music notation is only an approximate indicator. Use the offset tool to improve synchronization.
For the best result do the following:
1. Convert your music notation to a midi file using Musescore or similar program, name it the same as the midi file you want to load. (E.g. my_music.mid and my_music.xml)
2. Upload the midi file and the music notation file.
